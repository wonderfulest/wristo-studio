# 多选排列工具条美化设计

## 目标

优化右侧多选设置区的排列方式展示，解决标签与 6 个圆形按钮挤在同一行、操作层级不清晰的问题。

## 视觉设计

- “Align”表单标签独占一行，排列工具条在标签下方占满可用宽度。
- 6 个操作保持单行，按可用宽度等分。
- 按钮由独立圆形改为统一容器内的圆角方形分段按钮。
- 工具条使用面板边框、浅色背景和统一圆角，与右侧输入控件的视觉语言一致。
- 第 3、4 个按钮之间使用更明显的分隔线，区分前三个水平排列和后三个垂直排列。
- hover/focus 使用主题色浅底、主题色文字和焦点环；点击行为不增加持久选中状态。
- 每个按钮继续使用现有图标、翻译文本和原生提示标题。

## 实现边界

- 仅修改 `GroupSettings.vue` 的排列工具条结构类名和局部样式。
- 不改变 6 个操作的顺序、`alignSelection()` 调用、选区行为或历史记录。
- 不修改其他多选属性和底部编辑栏。

## 验证

- 右侧标题在工具条上方，6 个按钮等宽且不拥挤。
- 水平组与垂直组之间有清晰分隔。
- hover、键盘 focus 和明暗主题下均保持可辨识。
- 定向测试、全量单测和 `npm run build` 通过。
